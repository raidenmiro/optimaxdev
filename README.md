## Development

For development recommended use [asdf](https://asdf-vm.com/) with `pnpm`

- This application use svg icons that generates to sprite

```sh
pnpm sprite
```

- Use commitizen for commits

```sh
pnpm commit
```

- Installation deps

```sh
pnpm install
```

- Run local

```sh
pnpm dev
```

- `assets` - contains static icons for build it to sprite
